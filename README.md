# README - 체스택 데이터팩

## 설치 방법
1. 이 폴더를 마인크래프트 월드의 `datapacks` 폴더에 복사하세요.
2. 게임에서 `/reload`를 실행하세요.

## 사용 방법

### 게임 시작
```
/function chess:start
```

### 메뉴 열기
```
/function chess:menu
```

### 게임 리셋
```
/function chess:reset
```

## 게임 규칙

### 체스택 (Chesstack)
- 8x8 보드에서 진행되는 변형 체스 게임
- 각 플레이어는 킹으로 시작하며, 포켓에서 기물을 배치할 수 있습니다
- 포켓 점수는 39점으로 제한됩니다

### 스택 시스템
1. **스턴 스택**: 기물이 착수될 때 기물 점수만큼 부여됩니다
   - 스턴 스택이 1 이상이면 기물을 움직일 수 없습니다
   - 매 턴 종료 시 1씩 감소합니다

2. **이동 스택**: 한 턴에 몇 번 움직일 수 있는지 결정합니다
   - 1~2점: 5스택
   - 3~5점: 3스택
   - 6~7점: 2스택
   - 8점 이상: 1스택
   - 이동할 때마다 1 감소합니다

### 기물 점수표
- 폰: 1점
- 킹: 4점
- 나이트: 3점
- 비숍: 3점
- 룩: 5점
- 퀸: 9점

### 승리 조건
상대의 로얄 피스(킹)를 잡으면 승리합니다.

## Chessembly
이 게임은 Chessembly 언어로 기물의 행마법을 정의합니다.
자세한 내용은 `docs/chessembly/` 폴더를 참조하세요.
